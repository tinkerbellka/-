Я реализовала интерактивную игру "Монти Холл".
При запуске игры машина случайно размещается за одной из трёх дверей (эмулируется как JSON-объект внутри JS: { "prizeDoor": ... }).
Игрок выбирает дверь.
Открывается одна из оставшихся дверей с козлом.
Игроку предлагается оставить выбор или сменить.
На втором шаге отображаются все двери и показывается, выиграл ли игрок автомобиль или нет — на основе выбранной стратегии и фактического расположения машины.
